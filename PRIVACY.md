# Spoiler Shield Privacy Policy

Spoiler Shield does not collect, transmit, sell, or share any personal data.

All settings — topic names, subreddit patterns, title keywords, and toggle
states — are stored locally in your browser's extension storage. If you are
signed into Chrome, these settings are synchronized across your devices using
Google's `chrome.storage.sync` API; this sync is performed by your browser,
never by us.

The extension's only network communication is fetching the public, open-source
topic-pack catalog (static JSON files) from this GitHub repository when you
open its options page or click Add/Refresh there. No information about you is
sent with these requests beyond what any ordinary web request contains.

Reddit page content is read only locally, inside your browser, to decide which
posts to blur. It is never recorded, stored, or transmitted.

Contact: open an issue at https://github.com/ArpitKubadia/spoiler-shield/issues
