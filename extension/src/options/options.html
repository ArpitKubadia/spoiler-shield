Spoiler Shield

A topic blurs a post when its subreddit matches any pattern (* is a wildcard, e.g. *f1*), or its title/flair contains any keyword (whole words, case-insensitive; multi-word keywords match as a phrase). One entry per line. Toggle topics on/off from the extension's toolbar popup. Import merges: it only ever adds topics/entries (deduplicated) and never overwrites or deletes what you already have.
